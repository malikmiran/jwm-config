my jwm config file

    walpaper

for it to work correctly u need to install

    network manager
    nautilus
    volumeicon
    mjwm(build from https://github.com/chiku/mjwm)
    lxpolkit(ie polkit)
    libnotifyd
    xfce4-power-manager
    paprius icon theme
